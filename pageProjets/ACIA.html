    Projets Réalisés :


    1. Extraction de Contenu HTML vers JSON :

      Objectif : Introduire une fonction qui extrait le contenu de toutes les graines (seeds) dans le contenu HTML et le convertit en format JSON exploitable.

      • Récupérer toutes les pages HTML concernant les graines.
      • Établir une connexion avec le Chat Azure d'OpenAI.
      • Demander à ChatGPT de traiter les fichiers et de fournir une sortie JSON.
      • Transformer les données JSON en un format adapté à la base de données.
      • Établir une structure de base de données normalisée pour stocker ces données.

      Liens GitHub : Transformation de Données



    2. Système d'Analyse de Contenu et de Notation pour le Contrôle Qualité en Utilisant OpenAI et Azure :

      Description du Problème : Créer un système complet qui extrait les URL à partir d'un morceau de page Web, utilise l'API OpenAI via Azure pour générer des questions et des réponses pertinentes, et utilise une fonction de recherche pour noter l'exactitude des dix premiers résultats de recherche par rapport aux questions et réponses générées.

      1. Extraction des URL : Extraire le morceau en utilisant une requête sur la base de données.
      2. Génération Question-Réponse : Utiliser l'API Azure OpenAI pour générer des questions et des réponses correspondantes en fonction du contenu.
      3. Extraction de Morceau : Mettre en œuvre une méthode pour extraire des morceaux de texte à partir des données collectées.
      4. Jeu de Questions-Réponses Générées : Créer un ensemble de données de paires Question-Réponse à partir des réponses de l'agent conversationnel.

      Liens GitHub : Génération de Questions
